# Kelev Security SARL – Site vitrine (React + Tailwind)

## Démarrage

1. Installer les dépendances:
   ```bash
   npm install
   ```
2. Lancer le serveur de développement:
   ```bash
   npm run dev
   ```
3. Ouvrir l’URL indiquée par Vite.

## Logo

- Déposez votre fichier dans `public/logo.png` (transparent recommandé). Il sera affiché automatiquement.

## Personnalisation

- Couleurs principales: noir (`#0B0B0B`) et doré (`#C6A15B`).
- Mettez à jour les coordonnées dans `src/components/Footer.jsx`.
- Renseignez les numéros de téléphone/WhatsApp.
- Le formulaire de contact est prêt pour intégrer un service d’envoi (Email/API).
